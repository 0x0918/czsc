# chan - 缠论技术分析工具
>缠论来源于[缠中说缠博客](http://blog.sina.com.cn/chzhshch)

## 安装

Pypi上已经存在一个名为chan的库，以致于这个库没法上传到Pypi。

执行以下代码直接从github安装：
```
pip install git+git://github.com/zengbin93/chan.git
```
